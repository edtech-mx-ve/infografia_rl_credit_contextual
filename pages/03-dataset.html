Variables excluidas del contexto

Sí se utiliza en la práctica, pero no es visible para LinUCB durante la decisión.

contexto
→ agente decide
→ después se consulta loan_status
→ se calcula la recompensa

loan_grade

Se excluye porque incorpora una valoración histórica de riesgo y puede actuar como proxy de una evaluación crediticia previa.

loan_int_rate

No se incorpora al vector de contexto. Se utiliza como componente financiero auxiliar de la recompensa. Sus faltantes se imputan con la mediana calculada exclusivamente en entrenamiento.

La selección de variables no es mecánica: cada variable tiene un papel experimental específico.
